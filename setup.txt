Hi yall this is the setup for the react + asp.net

check if you have all the stuff you need through
dotnet --version
node --version
npm --version

then if you want to start a new env with react + net but not needed step only if you want to make you own project with react + net
then do 
dotnet new install Microsoft.DotNet.Web.Spa.ProjectTemplates

in ur terminal

after
run "dotnet new react -o ReactwithASP"

now understanding project set up

hmsproject
├── ClientApp               # Contains the React frontend code
├── Controllers             # Contains API controllers for ASP.NET Core(all the backend goes here)
├── obj                     # Build files and dependencies(is ignored)
├── node_modules
├── pages                   # Optional folder for MVC views (if needed)- but not
├── Properties              # Project properties
├── .gitignore
├── appsettings.Development.json
├── appsettings.json
├── Program.cs              # Main entry point for ASP.NET Core app
├── ReactWithASP.csproj     # Project file for ASP.NET Core
└── Startup.cs              # ASP.NET Core app configuration

do this in bash
dotnet add package Microsoft.AspNetCore.SpaServices.Extensions
dotnet dev-certs https --trust( if pop happen just click trust)
then to run is 

dotnet run
